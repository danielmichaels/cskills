# Generic skills and plugins

## code-review-graph

First this needs to be run if not already done:

```bash
claude plugin marketplace add tirth8205/code-review-graph
claude plugin install code-review-graph@code-review-graph
```

Then in the repo:

```bash
claude -p "Build the code review graph for this project"
```

This only needs to be done once, in every new project.
